# 📚 ProFlash - Flashcard Quiz App (CodeAlpha Task 2)

This repository contains the **"ProFlash"** project, developed as my second task during my **Web Development Internship at CodeAlpha**. The application is designed to help users study more effectively through interactive and customizable flashcards.

---

## 🚀 Project Overview

The **ProFlash App** is a modern, responsive, and interactive web application. It showcases advanced JavaScript concepts including **DOM manipulation**, **event handling**, and **localStorage integration**, combined with a clean UI and smooth user experience.

---

## ✨ Key Features

1. **Flashcard Flip Animation:** Seamlessly switch between question and answer.
2. **Easy Navigation:** Navigate through flashcards using Next and Previous buttons.
3. **Deck Management:** Add and delete flashcards for a personalized study experience.
4. **Persistent Storage:** Data is saved using localStorage, so it remains after reload.
5. **Keyboard Shortcuts:** Enhance usability with quick keyboard interactions.
6. **Responsive Design:** Optimized for both desktop and mobile devices.
