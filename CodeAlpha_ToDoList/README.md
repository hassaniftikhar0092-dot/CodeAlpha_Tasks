# 📝 Simple To-Do List - CodeAlpha Task 2

This repository contains the **"Simple To-Do List"** project, developed as my second task during my **Web Development Internship at CodeAlpha**. This application helps users efficiently manage their daily tasks by allowing them to add and remove tasks with ease.

---

## 🚀 Project Overview

The **To-Do List App** is a clean, responsive, and user-friendly web application. It demonstrates core JavaScript concepts such as DOM manipulation and event handling, combined with a simple and modern UI design.

---

## ✨ Key Features

1. **Add Tasks:** Quickly add new tasks to your list.
2. **Delete Tasks:** Remove tasks instantly with a single click.
3. **Input Validation:** Prevents users from adding empty tasks.
4. **Responsive Design:** Fully optimized for both mobile and desktop devices.
5. **Minimal UI:** Clean and distraction-free interface for better user experience.
